HammerTestApp  
Тестовое задание для Hammer Systems.  
    
Open API: Spoonacular API (доступен только title и image, поэтому в поля для описания ингридиентов и цены пришлось так же вставить title) + это англоязычный API, поэтому названия категорий тоже на английском (при использовании русскоязычного API, категории, соотвественно, также будут на русском языке).      
Technologies: PinLayout, Alamofire, SDWebImage. Добавлен опциональный offline-режим.
